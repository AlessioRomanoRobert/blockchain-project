# Blockchain Project


## Setup 🚀

Lo primero que hay que hacer es instalar yarn 

```
npm install --global yarn
```

iniciamos yarn

```
yarn init
```

Añadimos babel 

```
yarn add @babel/core @babel/node @babel/preset-env --dev
```

Añadimos nodemon 

```
yarn add nodemon --dev 
```

Añadimos express y body-parser 

```
yarn add express
yarn add body-parser
```

Añadimos web socket

```
yarn add ws
```

Añadimos elliptic para calculos matematicos

```
yarn add elliptic
```

Añadimos uuid para ids unicos

```
yarn add uuid
```

## Despliegue 📦

Añadimos los script en el package.json ("start": "babel-node index.js", "nodemon": "nodemon --exec yarn start") y después:

```
yarn nodemon
```
